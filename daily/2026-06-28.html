DeepSeek 发布 DeepSpec：Speculative Decoding 全栈框架
完整代码库用于训练和评估 speculative decoding 算法。核心思路是让小模型先"猜"大模型的输出，大模型只验证猜对的部分，从而加速推理 2-3 倍。这是当前 LLM serving 最热的优化方向之一。

关键细节：支持多种 draft model 架构，提供 benchmark 对比，包含推理引擎集成示例。对于做 LLM serving 的团队来说是直接可用的参考实现。

向量点积（Dot Product）
两个向量的点积 = 对应分量乘积之和。几何意义是衡量两个向量"方向有多一致"。
a · b = a₁b₁ + a₂b₂ + a₃b₃ = |a||b| cos θ
• 点积 = 0 → 两向量垂直（正交）
• 点积 > 0 → 方向大致相同（夹角 < 90°）
• 点积 < 0 → 方向大致相反（夹角 > 90°）
• 为什么重要：搜索引擎用点积算文本相似度，ML 里用点积做 attention 计算

📝 练习

1. 若 a = (1, 0)，b = (0, 1)，则 a · b = ？
✅ 正确！1×0 + 0×1 = 0。说明 x 轴和 y 轴是正交的。
❌ 再想想。1×0 + 0×1 = ？
❌ 不对。试试逐分量相乘再相加。
2. 若 a · b = 0，说明什么？
❌ 相等的话点积应该等于 |a|²。
✅ 正确！cos 90° = 0，所以点积为 0 意味着两向量正交（垂直）。
❌ 平行的话点积 = |a||b|，不为 0。
