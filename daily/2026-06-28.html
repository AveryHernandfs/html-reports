📝 今日一题：向量基础
题目
已知向量 a = (2, 3, 1)，b = (1, -1, 2)。
求：a · b（点积）的值。
答案：a · b = 2×1 + 3×(-1) + 1×2 = 2 - 3 + 2 = 1
今日概念
向量点积（Dot Product）
两个向量的点积 = 对应分量乘积之和。
几何意义：a · b = |a| |b| cos θ
如果点积为 0，说明两个向量垂直（正交）。
