今日概念 · 向量点积 (Dot Product)

向量点积（Dot Product）
两个向量的点积 = 对应分量乘积之和。几何意义是衡量两个向量"方向有多一致"——角度越小，点积越大。
a · b = a₁b₁ + a₂b₂ + a₃b₃ = |a| · |b| · cos θ
直觉：如果两个向量方向完全相同，点积 = 它们长度的乘积；垂直时点积 = 0；相反时点积 = 负数。

AI 中的应用：Transformer 里的 Attention 机制核心就是 Query 和 Key 的点积——衡量两个向量的"相关度"。
